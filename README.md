# Projeto Final de Angular

Nome do pojeto: Study Manager
Aluna: Juliana Quintas, iefp256

## Funcionalidades
- dashboard que permite visualizar rapidamente as tarefas por completar com prioridade mais alta, e as que têm de ser entregues no dia actual;
- adicionar disciplinas personalizada, definindo o nome e número de créditos;
- adicionar tarefas associadas a uma descrição, disciplina, data de entrega e prioridade;
- marcar tarefas como completas (arquivando-as) ou incompletas (retira-as do arquivo);
- filtro de tarefas por disciplina ou se estão arquivadas.
